Deliverable 01 · Tokens

Warm paper, terracotta, one editorial serif.

Colors are oklch with chroma capped at 0.012 on neutrals so the warmth reads as paper, not as a color. Dark mode is a deep warm slate with the same hue, never flat black. A single accent — terracotta — carries primary CTAs and focus rings. Type pairs Inter for UI with Source Serif 4 for the one editorial moment per screen (rendered post blocks).
